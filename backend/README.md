# Backend e Frontend Laravel (estrutura)

Este diretório contém a estrutura para rodar o projeto no padrão Laravel full stack:

- API admin para autenticação e CRUD
- Frontend em Blade (HTML + CSS + Bootstrap)

## Rotas Web (frontend Laravel)

- `/` Home
- `/produtos` Catálogo
- `/produto/{slug}` Página de produto
- `/cidades` Lista de cidades
- `/cidade/{slug}` Landing SEO por cidade
- `/depoimentos`
- `/contato`

## Rotas API (admin)

- `POST /api/login`
- `cities` CRUD
- `products` CRUD
- `testimonials` CRUD
- `GET/PUT /api/settings`
